### 1.0.1.0 (2023. 06. 14.)

#### 기능 개선/변경

* 런처 셀프 업데이트 기능을 별도의 업데이터에서 PowerShell로 변경함.
* 관리자 권한이 필요한 경로에 대한 설치 경로 설정을 제한함(AppData/Roaming/ 하위만 설치).
* 게임 실행 실패 시 예외를 메시지 창으로 표시함.
* 내부 구조를 개선함.

#### 버그 수정

* 게임 설치 경로를 관리자 권한이 필요한 곳으로 설정할 때 발생하는 오류를 수정함.
* 런처에서 게임 실행 실패 예외를 사용자에게 알리도록 수정함.

### 1.0.0.0 (2022. 01. 25.)

#### 신규 상품 출시

* 게임 클라이언트를 다운로드/설치/실행할 수 있는 기능을 제공하여 안정적으로 게임을 운영하고 실행할 수 있도록 합니다.
    * 런처를 설치할 수 있는 인스톨러를 제공합니다.
    * 웹페이지에서 PC에 설치된 런처를 실행할 수 있는 기능을 제공합니다.
    * NHN Cloud 콘솔을 통해 런처를 설정하고 게임을 관리할 수 있습니다.
